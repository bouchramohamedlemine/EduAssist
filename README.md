# EduAssist


## Setup Instructions

### 1) Supabase
1. Create a new Supabase project.  
2. Update the project with your documents:  
   - Open the `upload_docs` folder in the project.  
   - Follow the instructions in the `README` file inside `upload_docs` to upload your documents to Supabase.  

### 2) Next.js
1. Navigate to the `eduassist` folder (the Next.js project).  
2. Install dependencies:  
   npm install
Run the development server:
   npm run dev



Open the app in your browser and start asking questions.

**Instructions about the Next.js app are in the README file inside the eduassist folder.**
